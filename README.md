# fake-news-detector
Detekcja Fake Newsów przy użyciu przetwarzania języka naturalnego (NLP). Projekt polega na analizie tekstów wiadomości oraz zbudowaniu modelu klasyfikacyjnego, który rozróżnia wiadomości prawdziwe od fałszywych.

# Cel
Pokazanie, że fake newsy mają charakterystyczne wzorce językowe, które można wykrywać automatycznie.

# Technologie
- Python (pandas, re, NLTK, scikit-learn, matplotlib)
- Jupyter Notebook (Google Colab)

# Jak uruchomić
1. Biblioteki (pip install pandas nltk scikit-learn matplotlib)
2. Pobierz pliki True.csv i Fake.csv z linku w pierwszej komórce i umieść je w folderze z plikiem notebook.
3. Uruchom komórki

# Czego się nauczyłem
- label'ingu danych
- czyszczenia tekstu z artefaktów
- tokenizacji i stopwordów
- analizy ekploracyjnej i identyfikacji słów unikalnych dla fake newsów
- modelu Regresji Logistycznej w Machine Learning
